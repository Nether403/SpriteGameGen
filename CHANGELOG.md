## [Unreleased]

### Added

- Initial project governance, contribution, security, and issue-reporting documentation.
- Manifest V2 with retained source assets, deterministic quality/repair controls, and multi-clip character workspaces.
- Deterministic frame archives and versioned character bundles with a Godot 4.7 AnimatedSprite2D profile.
- Strict data-only action packs, credential-free recipes, the `sprite` CLI, and resumable batch state.
- Optional capability-aware loopback ComfyUI provider with operator-owned workflows.
- Expanded direct MCP tools for clips, repair, bundles, and read-only recipe operations.
